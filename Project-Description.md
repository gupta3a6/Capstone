Project topic: A sublease application to help students find sublease or roommates to live with off campus.

FACULTY ADVISOR: Dr. Will Hawkins

Abstract: Sub4You is a web-based platform designed to connect individuals looking to sublease their housing with those seeking temporary accommodations. The system streamlines the process through profile matching, request workflows, and a modern interface, improving efficiency and trust.

Description: Sub4You addresses the inefficiencies in the subleasing market, particularly for students.

Key Features:
Dual user roles: Listers & Seekers
Profile-based matching system
Request/accept workflow
Clean, modern UI
Scalable architecture design
